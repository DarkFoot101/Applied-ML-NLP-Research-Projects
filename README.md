### 3Ô∏è‚É£ Real-Time Facial Expression & Behavioral Analysis System

A **real-time facial expression recognition system** focused on ethical, analytical, and behavioral emotion analysis in unconstrained environments.  
This project is backed by **published IEEE research** and includes extended experimental work beyond the paper.

#### Research Publication
**‚ÄúAI Pioneering Ethical, Analytical, and Real-Time Emotional Recognition in Dynamic Human Expressions‚Äù**  
- Published in **IEEE ‚Äì ICCCD-SAI 2025**  
- DOI: `10.1109/ICDSAAI65575.2025.11011864`

#### Key Features
- Real-time facial expression detection using CNNs
- Behavioral and malpractice detection (unauthorized objects)
- Visual highlighting of suspicious activity
- Automated monitoring support for proctoring scenarios
- Achieved **83% accuracy on FER-2013 dataset**

#### Tech Stack
- Python  
- TensorFlow, Keras  
- OpenCV  
- FER-2013 Dataset

### 4Ô∏è‚É£ Book Recommendation System

A **collaborative filtering‚Äìbased book recommendation system** providing both **popularity-based** and **personalized** recommendations, deployed as a web application.

#### Key Features
- Top 80 books using popularity-based filtering
- Personalized recommendations using cosine similarity
- Trained on **10,000+ user‚Äìitem interactions**
- Flask-based backend with interactive UI

#### Tech Stack
- Python  
- Flask  
- Scikit-learn  
- Pandas, NumPy  
- HTML, CSS, Bootstrap  
- Pickle (model persistence)

Dataset:  
[Kaggle Book Recommendation Dataset](https://www.kaggle.com/datasets/arashnic/book-recommendation-dataset)
